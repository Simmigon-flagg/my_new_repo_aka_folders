How to get code from GITHUB onto my local computer

  1. Create a new repo
  2. Select public or private
  3. Click the green code button
  4. select SSH and copy the URI
  5. Go to your local terminal and type git clone "paste your URI"
  6. git clone git@github.com:Simmigon-flagg/my_new_repo_aka_folders.git

how to get code from your local computer into GITHUB

  1. optional: git status
  2. git add . or git add "type the file name"
  3. optional: git status
  4. git commit -m"First commit" -m"Optional"
  5. git push origin main
